# tv-series-navigation-repository
该仓库专门记录电视剧类站点在导航资料仓体系与页面层级方面的公开资料，通过文字归档剧集分类规则、年份分区逻辑与栏目层级组织形式，形成长期结构研究样本。项目不涉及影视资源本身，仅对公开页面信息进行整理说明。内容重点分析剧集页面在标签布局与路径设计方面的结构特征，用于页面组织形式研究。所有资料来源于公开页面，仅供信息研究与结构分析参考。
